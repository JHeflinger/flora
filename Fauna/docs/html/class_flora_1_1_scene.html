static void DestroyEntity (Entity entity)
 Destroys an Entity in the scene context. Note: this is the only way to properly destroy an entity. Destroying an entity object typically will just remove it from the script engine; the entity will still exist in the scene context unless you use this method to touch the Flora backend and delete the entity.
