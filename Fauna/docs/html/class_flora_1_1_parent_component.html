Detailed Description

The parent component of an entity. Stores which entity is the parent of this entity. Note: one can think of the subentity concept as a doubly linked n-ary tree. Each node can have multiple sub-nodes, and all sub-nodes will point toward their parent node. It is recommended to use functions such as Entity.AddChild(Entity child) to handle parent-child relationships, as unexpected outcomes may occur with the misuse of Parent and Child components.
