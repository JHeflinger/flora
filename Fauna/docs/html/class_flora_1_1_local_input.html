Detailed Description

This class is a local definition of the Input class. This class is specifically geared for Key and Mouse release. This is because the engine stores key and mouse activity on a frame-by-frame basis, so attempting to read when an input is released can cause inaccurate outputs. To fix this, use this local class for each time you wish to check for key and mouse releases.
